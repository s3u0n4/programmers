# 코드프로그래머스 - 동영상 재생기


처음에 video_len을 동영상의 길이를 쪼개서 구하려고 생각해서 반복문을 돌리려고 함 근데 반복문을 돌려야할건 command 배열이라는걸 알게 됨


그리고 시간도 분과 초로 구하려다가 안되는 걸 알게 되고 초로만 계산하고 나중에 분으로 바꾸기로 함 


어떤 걸 구해야 하는지 정하고 순서도로 나열해서 어떤게 우선순위인지 알게 되는 것이 지금 내 상태에선 힘든것 같음


처음 내 코드에서 고쳐야 할 점은 


반복문 → commands 기준


시간 → 전부 초로 변환


prev / next → 명령어마다 처리


오프닝 체크는 명령 전·후


그리고 padStart,padEnd 라이브러리도 알게되었다.


이 문제의 핵심은 분:초를 초로 바꾸는 로직이 제일 중요하다는 것을 알게 되었다.

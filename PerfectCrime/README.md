# 프로그래머스 - 완전범죄

처음으로 2단계 문제를 풀어보았고 문제를 이해하려고 많은 시간을 할애하였다.

문제를 이해하고 어떻게 풀어야하는지 머리로는 이해를 했는데 코드를 통해서 푸는 순서를 정하는 것은 하지 못하였다.

dp = Dynamic Programming(동적 계획법)을 이 문제에서 활용하였다. 말 그대로 풀면 계산한 결과를 기억하면서 문제를 푸는 방법이라고 한다.

A흔적 B흔적 상태를 기억하면서 가능한 상태만 다음으로 넘기는 방식으로 푼다.

점점 알고리즘을 활용하여 문제를 설계하는 부분이 어렵다고 느껴지는데 문제를 많이 풀면서 푸는 방식에 익숙해지는 시간을 가져야겠다.

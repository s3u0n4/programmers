# 프로그래머스 - 달리기 경주

이 문제를 풀면서 map을 사용해서 문제를 푸는 방법을 알게 되었으며

map에 관련된 map.set(),map.get() 함수에 관해 알게되었다.

치환하여서 문제 푸는 방식이 이해가 안됐는데 앞사람을 따로 변수에 저장하여 뒷사람을 앞으로 이동시키고 앞사람을 뒤로 이동시켜서 푸는 방법이었다.

한 줄로 풀게 되면 앞사람의 정보가 사라지기 때문에 변수에 저장하고 푸는 방식으로 풀어야 된다고 알게되었다.

다른 방식으로는 [players[idx - 1], players[idx]] = [players[idx], players[idx - 1]] 이렇게 푸는 방식이 있다.

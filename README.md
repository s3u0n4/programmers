# 코드프로그래머스 - 겹치는 선분의 길이 


처음 문제를 봤을 때 코드로 구현하려하니 도저히 감이 잡히지 않아 다른 사람들의 문제 풀이를 보았다.


다른 건 다 이해가 되었는데 


let start = lines[i][0]; //선분 시작 


let end = lines[i][1]; // 선분 끝


이 부분이 이해가 되지 않았다. 


그래서 이 부분을 chatgpt 한테 물어서 자세한 예시로 들며 하나하나 뜯어서 이해하려고 하니까 이해가 되기 시작했다.


종합적으로 선분을 하나하나 돌면서 돌았던 부분이 또 나오면 +1을 하면서 겹친 구간이 또 나오면 겹친 횟수을 겹친 구간의 길이로 바꿔서 구하는 문제인 것 같다.


배열의 이해도가 더 필요할 것 같고 문제의 지문을 하나하나 뜯어서 코드 구현을 하는 연습을 더 해야할 것 같다고 느꼈다.

for (let x = i - 1; x <= i + 1; x++), 


for (let y = j - 1; y <= j + 1; y++),

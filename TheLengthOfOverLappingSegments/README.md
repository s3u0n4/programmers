# 코드프로그래머스 - 안전지대

0단계 문제를 풀면서 가장 어려웠던 문제였던 것 같다. 처음 볼 때는 이 문제가 무엇을 말하고 있는지 어떻게 풀어야할지 감도 안 잡혔다.

이 문제 또한 배열 관련 문제인데 배열의 유효성 검사를 할때 특히 이해가 되지 않은 것 같았다.

for (let x = i - 1; x <= i + 1; x++),

for (let y = j - 1; y <= j + 1; y++),

if (x >= 0 && x < n && y >= 0 && y < n), 이 부분들이 이해가 되지 않아 여러 번 돌려본 것 같다.

문제를 풀면서 제한 사항도 체크해야하고 반복문으로 배열을 만드는 것도 연습을 많이 해야할 것 같다.
